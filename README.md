
개인 제작물 기록 과정에서 콘텐츠 분류, 시각적 식별, 반응형 구조를 검증하기 위해 제작한 웹사이트의 소스들을 저장하는 장소입니다.

이 사이트의 차별점은 LLM언어모델을 통해 아카이브형 웹사이트를 제작할 수 있다는걸 보여주기 위함입니다.

################################################################################

Ⅰ.사용 기술 스택

[Frontend]

	-HTML5
	-CSS3
		①Flexbox 기반 레이아웃
		②Dark / Light Mode 지원
	-Vanilla JavaScript
		①카테고리 필터링
		②이미지 갤러리 렌더링
		③사이드바 토글
		④다크모드 상태 제어
		⑤페이지 내 인터랙션 처리

[Backend]

	-Node.js
	-Express
	-Multer
		①이미지 파일 업로드 처리
		②서버 디렉토리 저장
---
Ⅱ.주요 기능

[이미지 업로드 & 자동 관리]

	-서버로 이미지 업로드
	-카테고리별 폴더 구조로 저장
	-클라이언트에서 자동 로딩

[이미지 갤러리 & 미리보기]

	-카테고리 / 서브카테고리 분리
	-페이지네이션
	-이미지 투명도 조절 기능
	
[UI / UX 요소]

	-다크 모드 토글
	-사이드바 메뉴
	-로딩 인디케이터
	-반응형 레이아웃
	
[인터랙션 콘텐츠]

	-숨겨진 기능 페이지
	-웹 미니 게임
	-아이콘 기반 진입 구조
---
Ⅲ.모바일 앱 연동 (웹 게임 기반)

본 프로젝트에는 웹에서 제공되는 미니 게임 기능이 포함되어 있으며, 해당 웹 게임을 WebView 기반 모바일 앱으로 확장하여 출시할 예정입니다.

	-웹과 모바일 앱은 동일한 게임 로직을 공유
	-모바일 환경에 맞게 UI 및 조작 방식 최적화
	-앱 전용 기능 및 게임 모드 추가

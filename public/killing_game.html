킬링 게임 페이지

이미지를 클릭하여 게임을 시작하세요.

힌트 내용
찾은 숫자:

축하합니다! 모든 숫자를 찾았습니다!

해당 팝업에서 "확인"버튼을 클릭한 후 이미지를 클릭하면 게임대기화면으로 이동합니다.

게임 조작법

*스페이스바 ==> 숫자 생성

*방향키(↑,↓,←,→) ==> 객체 이동

*R키 ==> 객체 회전

*H키 ==> 다음숫자 위치보기

space바를 누르면 숫자가 전부 사라지기전까지 키보드 조작을 할 수 없습니다.

게임 방법

▶space바를 누른후 랜덤한 위치에 1~10까지의 숫자를 생성한후 점점 옅어지면서 사라집니다.

▶숫자가 전부 사라지면 "게임시작 문구"가 나온 뒤에 이동하면서 숨겨진 숫자들을 찾으시면 됩니다.

▶(중요)객체를 누르면 다시 조작법과 게임방법을 알 수 있습니다.

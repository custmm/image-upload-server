Q:이 웹사이트를 소개해주세요.
A:이 웹사이트는 제작자(본인)가 취미비용을 사용하여 제품을 구입하고 가정에서 조립한후 완성품을 사진으로 남겨서 간직하기 위한 사이트입니다.

Q:이 웹사이트에 특이한 기능은 무엇인가요?
A:그냥 보고 넘기는 방식에서 지루함을 덜어내기위하여 특이한 아이콘, 특수효과, 숨겨진게임, 어서오세요를 누르면 각 항목의 소개가 나오고 체험할수있는 공간이 마련됩니다.

웹 게임과 앱게임의 차이
* 웹 게임 게임 앱
진입경로 표시기 아이콘을 라이트/다크모드 토글 위에 올려서 진입 어플 시작
게임구성
게임대기화면 팝업에서 "확인" 클릭 후 진입
게임시작 스페이스바(space) 누름과 동시에 숫자가 옅어짐
게임모드 선택 모드선택 없음
웹페이지에서 나오는 랜덤한 아이콘으로 시작
게임대기화면 하고싶은 아이콘 선택 후
게임 관련 팝업에서 확인 누른 후 진입
게임시작 "GAME START" 버튼 터치와 동시에 숫자가 옅어짐
★모바일 앱 게임은 더 많은 게임 모드와 자유로운 커스터마이징으로 웹 게임보다 더 다양한 플레이가 가능합니다!

게임앱에만 있는 기능

1)게임모드
태초마을이야!(일반 모드) 복불복게임(랜덤 모드) 내 맘대로 하기(커스텀 모드)
웹 게임과 비슷하지만 아이콘 선택 가능 ★웹 게임에는 없는 모드라서 다양한 선택이 가능
주사위 아이콘으로 난이도와 아이콘을 랜덤 결정 찾을 숫자 수, 체력 소모 시간, 아이콘을 자유롭게 커스터마이징 가능

2)추가기능
제외할 아이콘 선택 플레이 하기 싫은 아이콘을 선택해서 제외시킬 수 있음
테마 설정 앱 내의 알림창과 설정창 "테마","게임 아이콘 스타일" 등을 변경할 수 있음
세부 설정 게임 배경화면 변경 앱 내의 배경색상을 변경할 수 있음
(특정 배경화면색을 통해 히든 모드로 갈수있는 통로로 갈 수있음)
게임 컨트롤러 수정 앱 내에서 게임 아이콘의 움직임을 제어하며, 컨트롤러의 높이와 컨트롤러의 투명도를 조절할 수 있음
